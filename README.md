# Crypto Legal Skill

A Solana AI Kit skill that helps founders navigate crypto regulations, compliance, and incorporation.

## What it does
- Answers common questions about stablecoin regulations in different countries.
- Provides guidance on legal entity setup for crypto projects.
- Suggests compliance steps based on user's jurisdiction.

## Installation
1. Clone this repo into your `solana-ai-kit/skills/` folder.
2. Run `./install.sh` (if provided) or just use the skill via Claude Code / Codex.

## How to use
Simply ask: "What are the stablecoin regulations in [country]?" or "How do I incorporate a DAO in the US?"

## Requirements
- Python 3.8+
- Access to the Solana AI Kit (Claude Code / Codex)
